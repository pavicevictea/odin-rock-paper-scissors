# Odin Rock, Paper, Scissors Game

This project is part of **The Odin Project** curriculum and is an interactive implementation of the classic **Rock, Paper, Scissors** game built with **HTML**, **CSS**, and **JavaScript**.

The main goal of this project was to practice **DOM manipulation**, **event handling**, and **game logic implementation** in JavaScript.  
It allows a player to compete against the computer, with live score updates and a simple UI that displays round results and a final winner.

## Skills Demonstrated
- Writing clean and semantic HTML
- Styling and layout with CSS
- Implementing game logic using JavaScript
- Using DOM manipulation to update the UI dynamically
- Handling events and managing state in a browser-based game
- Adding basic interactivity for improved UX

## Features
- Play **Rock, Paper, Scissors** against the computer
- Real-time **score tracking**
- **Color-coded** round results (Win / Lose / Tie)
- Game ends when either side reaches **5 points**
- **Play Again** button to restart the match
